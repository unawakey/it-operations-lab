# IT Operations Lab

This repository documents my journey of building an IT Operations homelab to better understand enterprise IT concepts such as Windows Server, Active Directory, and virtualization.

## Goals

- Learn Windows Server administration
- Configure Active Directory Domain Services (AD DS)
- Understand DNS in a domain environment
- Create and manage users, groups, and Organizational Units (OUs)
- Join Windows clients to a domain
- Practice documenting my setup and troubleshooting process
- Build a strong foundation in IT Operations and Systems Administration

## Lab Environment

**Host Machine**

- Debian 13 (Trixie)
- ThinkPad X230
- Intel Core i7-3520M
- 16 GB RAM
- KVM/QEMU with Virt-Manager

**Virtual Machines**

- Windows Server 2022
- Windows 11 Pro

## Disclaimer

This is a personal learning project. I'm building this lab to gain hands-on experience with enterprise IT technologies and document what I learn along the way.
